# Unmanned Parking Reservation System - Extended Version

C 기반 무인 주차관리시스템 확장 버전입니다. 기존 예약/입차/정산 흐름에 예외 처리, 예약 변경/취소, 관리자 수정, 매출 통계, 전자 영수증, 자동 새로고침, 예약 만료 스레드를 추가했습니다.

## 추가된 주요 기능

1. 전화번호 형식 검사 및 자동 하이픈 저장
   - `01012345678`로 입력해도 `010-1234-5678`로 저장
   - `010-****-****` 형식만 허용

2. 차량번호 형식 검사
   - `12가3456`, `123가4567` 형식 허용
   - 숫자 2자리 또는 3자리 + 한글 1자리 + 숫자 4자리

3. 예약 순서 변경
   - 차량번호/전화번호 입력 후 주차타워를 먼저 선택하고 차종을 선택

4. 예약 만료 자동 삭제
   - 예약 후 20분 안에 입차하지 않으면 예약 레코드를 실제로 삭제
   - `pthread` 기반 백그라운드 스레드가 10초마다 만료 검사

5. 관리자 수정 기능
   - 예약의 입차시간, 출차시간, 요금, 타워, 차종, 상태 수정 가능
   - 결제 기록의 주차요금, 최종 정산금액, 결제방식 수정 가능

6. 요금계산기 날짜 선택
   - `YYYY-MM-DD HH:MM` 형식으로 입차/출차 날짜와 시간 입력

7. 예약 취소 기능
   - 예약 완료 상태에서 사용자가 직접 취소 가능

8. 매출 통계 기능
   - 전체 매출, 평균 결제금액, 타워별 매출, 차종별 매출, 결제수단별 매출 출력
   - 웹 연동용 CSV를 `data/sales_report.csv`에 생성

9. 실시간 주차 가능 대수 자동 새로고침
   - 새로고침 간격과 횟수를 입력하면 주차 가능 대수를 반복 출력

10. 차종 기반 최적 주차타워 자동 추천
    - 해당 차종의 잔여 대수가 많고 혼잡도가 낮은 타워 추천
    - 예약 중 추천 타워로 자동 배정 선택 가능

11. 주차타워 운영시간 정보 추가
    - 기본 운영시간: 24시간 운영

12. 전자 영수증 발급
    - 정산 후 영수증 번호 자동 생성
    - 영수증 정보 출력 및 `data/receipts.txt` 저장

13. 예약 변경 기능
    - 주차타워와 차종 변경 가능
    - 변경해도 예약 제한시간 20분은 새로 늘어나지 않음

14. 예약번호 찾기
    - 차량번호 + 전화번호 인증 후 활성 예약번호 확인

15. 예약 제한시간 카운트다운
    - 예약 조회 창에서 남은 입차 제한시간 표시

16. 입차 내역 조회
    - 입차시간, 입차 경과시간, 현재 예상요금, 보증금 차감 후 예상 정산금액 표시

## Build

```bash
make
```

## Run

```bash
./parking_system
```

## Reset Data

```bash
make reset-data
make
./parking_system
```

## Admin Password

```text
admin1234
```

## Data Files

- `data/towers.dat` : 주차타워 정보
- `data/reservations.dat` : 예약 정보
- `data/payments.dat` : 결제 정보
- `data/log.txt` : 시스템 로그
- `data/messages.txt` : 예약번호 발송 기록
- `data/receipts.txt` : 전자 영수증 기록
- `data/sales_report.csv` : 웹 연동용 매출 CSV

## Notes

- 예약 만료 스레드를 사용하므로 Makefile에 `-pthread`가 포함되어 있습니다.
- 차량번호 검사는 UTF-8 한글 한 글자를 기준으로 처리합니다.
- 기존 `.dat` 파일 구조와 맞지 않으면 프로그램 시작 시 해당 데이터 파일이 초기화될 수 있습니다.
